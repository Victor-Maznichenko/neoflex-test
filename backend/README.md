# API сервер для интернет-магазина QPICK

Express.js сервер для технического задание в Neoflex.

## Технические требования

- Node.js v20+
- Express.js

## Установка и запуск

```bash
# Переход в директорию сервера
cd qpick-backend

# Установка зависимостей
npm install

# Запуск сервера
npm start

# Или запуск в режиме разработки с автоматической перезагрузкой
npm run dev
```

Сервер будет доступен по адресу: `http://localhost:3001`

## Документация API

API задокументировано в файле `schema.yaml` в формате OpenAPI 3.0.

### Версионирование API
API использует версионирование через URL path: `/api/v1/`
